Return probability distribution for angel investors

Introduction

Below graphs depict probabilities for angel investor portfolio outcomes. The graps are based on a simplified model. In this model it is assumed that the investor has a portfolio with a specified number of investments. Further it is assumed that each investment has a fixed probability to either be a success or be a complete failure.

Each success has a fixed multiple return . The model also assumes a fixed number of year before successful investments exit. Finally, the model assumes that the same amount is invested in all portfolio companies.

You can select your own parameters. Respectively, the graphs below show proability distributions for the:

  • number of successes
  • aggregated portfolio multiple
  • aggregated portfolio internal rate of return (IRR)

Although this model is a simplification of reality, it does provide insights in what investment strategy to follow, depending on the investors risk appetite and expectations.
